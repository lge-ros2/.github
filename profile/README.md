# 🤖 LGE-ROS2

> Advancing robotics through ROS 2, simulation, and AI.

LGE-ROS2 is an open-source robotics initiative focused on building scalable robot software platforms, simulation technologies, and data infrastructure for next-generation intelligent robots.

### Core Areas

- ROS 2 Platform
- CLOiSim Multi-Robot Simulator
- Cloud & Edge Robotics
- Robot Learning Infrastructure
- Navigation & Autonomy
- AI-powered Robotics

### Featured Projects

- **CLOiSim** — Unity-based multi-robot simulation platform
- **cloisim_ros** — ROS 2 integration layer
- **cloi_common_interfaces** — Common robot interfaces
